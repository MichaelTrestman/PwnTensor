GameMind Subnet — NPC/Bot AI Marketplace on Bittensor
Here’s the full design, thinking through it the way Chi intends: you describe the subnet idea, then the knowledge folder tells an agent exactly how to build validator.py and miner.py.

The Core Concept
You are the subnet owner / customer. You bring a game (say, an FPS, an RPG, a strategy game). You want NPCs that feel alive — good pathing, smart dialogue, tactical behavior, whatever your game needs. You post that game as a task environment on the subnet. Miners compete to produce the best-performing bot/agent for your game. Validators judge who wins. TAO emissions flow to the best performers.
The subnet is essentially a continuous AI performance tournament where the commodity being produced is game AI behavior.

Roles Mapped to Bittensor Primitives
Miners = bot/agent developers. They register a hotkey and run an agent that responds to game state queries. Under the hood they can use anything — a hand-crafted finite state machine, a fine-tuned LLM, a reinforcement-learned policy, behavior trees, whatever. The subnet doesn’t care. It only scores outputs.
Validators = game environment runners. Each validator applies the subnet’s incentive mechanism to score miners’ performances, then submits those weights to the Bittensor blockchain. ￼ In your case, the validator spins up the game environment (headless), sends game state to each miner’s axon, receives an action/response, steps the simulation, and scores the result.
Subnet Owner = you (or the game studio). You define the scoring rubric per game and register it as a sub-task on the subnet.

The Protocol (What Goes in validator.py and miner.py)
Request payload (validator → miner):

{
  "game_id": "dungeon_crawler_v2",
  "state": { "player_pos": [x,y], "npc_hp": 80, "nearby_enemies": [...], "dialogue_trigger": "greeting" },
  "context_window": [...last N ticks...],
  "timeout_ms": 200
}


Response payload (miner → validator):

{
  "action": "move_to",
  "target": [x, y],
  "dialogue": "Well met, traveler.",
  "emote": "wave"
}


The miner’s axon is always-on, serving these requests. The miner can be stateful (keeping a rolling context) or stateless.

The Incentive Mechanism — This Is the Clever Part
Each game defines its own scoring rubric, but validators aggregate across all registered games. A miner who’s great at one game but mediocre at another gets proportional credit.
Per-game score components:



|Dimension                 |How it’s measured                                                                         |
|--------------------------|------------------------------------------------------------------------------------------|
|**Behavioral correctness**|Did the NPC achieve its objective? (reach goal, survive, deal damage, complete quest step)|
|**Naturalness**           |Human rater panel OR an LLM judge scoring dialogue/movement for believability             |
|**Latency**               |Response under timeout gets full credit; degrades linearly after                          |
|**Consistency**           |Does the NPC behave coherently across a 30-tick episode? (no random pivots)               |
|**Adversarial robustness**|Edge cases — does it break when given weird state?                                        |

Aggregation across games:
The subnet maintains a registry of active game tasks. Each game task has a weight (set by the game owner staking TAO into that task slot — this is your monetization hook). The final miner score is:

score_miner = Σ (game_weight[g] × normalized_score[miner, g]) for all g


This means a game studio that wants better AI stakes more TAO into their game slot, which raises the reward signal for miners who perform well on their game specifically. Market-driven prioritization.

The Sub-Mechanism Per Game (Your Key Innovation)
Each game is essentially a mini-subnet inside the IM. The subnet owner can register a game with:
	∙	A game adapter (a Docker image or API endpoint the validator calls to run episodes)
	∙	A scoring function (defined in a game-specific config JSON)
	∙	A stake weight (how much TAO the game owner locks to prioritize this game)
	∙	A context schema (what state fields exist in this game)
Validators pull the game registry, run episodes in parallel across miners, and compute weighted scores. The Chi template’s validator.py becomes a loop that iterates over active game tasks each epoch.

What Miners Actually Build
The beauty of this design is that miners are implementation-agnostic. Their only contract is: receive a game state JSON, return an action JSON within the timeout. So miners can build:
	∙	A scripted bot for a specific game (cheapest to run, wins if the script is clever)
	∙	A fine-tuned small model (e.g., a 7B model trained on gameplay data)
	∙	A general-purpose game agent (like an LLM prompted with game context) that spreads across many games and picks up the long tail of game weights
	∙	A hybrid — script for the games they know, LLM fallback for unknown games
This is exactly the competitive pressure you want. Lightweight clever solutions can beat heavyweight AI if the scoring is tight enough, which keeps the compute costs reasonable for miners without GPUs.

How the Chi Template Gets Built Out
Following Chi’s pattern — clone into Cursor, point at @knowledge — the key files are:
validator.py — pulls game registry from chain storage, for each game fetches state from the game adapter, fans out requests to all miner axons, collects responses, runs scoring functions, aggregates weighted scores, submits weights every epoch.
miner.py — registers an axon, exposes a forward() function that receives GameStateRequest and returns NPCActionResponse. The miner’s intelligence lives entirely inside forward().
protocol.py — defines the synapse: GameStateRequest and NPCActionResponse as Pydantic models, versioned per game schema.
game_registry.py — a utility that reads the on-chain game task list and fetches game adapter configs.

Open Design Questions Worth Gaming Out Further
	1.	Who runs game adapters? The game studio has to expose a headless sim endpoint. This is a real technical ask — do you provide a standard SDK? Or do validators run the Docker image locally?
	2.	LLM judge vs. deterministic scoring — dialogue quality is hard to score without an LLM judge, but that introduces validator variance. Chi-style you’d want validators to agree closely or Yuma Consensus penalizes them.
	3.	Latency vs. quality tradeoff — 200ms is tight for an LLM call. You may want a separate “real-time” track (scripted/small model) and “async” track (heavyweight AI, for non-real-time NPC behaviors like quest dialogue).
	4.	Anti-gaming — miners could overfit to validator scenarios. You’d want validators to randomize episode seeds and occasionally inject synthetic edge-case states.

The Synapse (protocol.py)
Subnet validators and miners use Synapse data objects as the main vehicle to exchange information. The Synapse class inherits from Pydantic’s BaseModel. The validator sets immutable fields; the miner fills a mutable response field; the validator reads it back. ￼

class GameStateSynapse(bt.Synapse):
    # Set by validator — immutable
    game_id: str                   # "pacman" | "halo" | "mortal_kombat"
    episode_seed: int              # deterministic RNG seed for the match
    tick: int                      # current sim step
    state: dict                    # full game state snapshot (game-specific schema)
    valid_actions: list[str]       # legal moves for this tick

    # Filled by miner — mutable
    action: Optional[str] = None   # chosen action
    action_args: Optional[dict] = None  # e.g. {"target": [x,y], "move": "up"}
    latency_budget_ms: int = 150   # hard SLA per tick


One synapse type for all three games — game_id routes the miner’s internal logic. Miners that can handle multiple games get more scoring opportunities.

The Scoring / reward.py
The scoring model dictates how the task should be done, and should mimic user preferences or desired outcomes — like any ML objective function that the subnet continuously optimizes. ￼
For each game tick, the validator runs the deterministic sim forward using the miner’s returned action, then scores at episode end:

def score_episode(game_id, episode_log, miner_uid):
    if game_id == "pacman":
        # ghost catches pac = +1.0 per catch, normalized by episode length
        # pac escapes full board = -0.5 penalty
        raw = catches / max_possible_catches
    elif game_id == "mortal_kombat":
        dmg_dealt = sum(episode_log["p2_damage_events"])
        dmg_taken = sum(episode_log["p1_damage_events"])
        raw = (dmg_dealt - dmg_taken) / MAX_HEALTH  # normalized -1 to +1
    elif game_id == "halo":
        kills = episode_log["kills"]
        deaths = episode_log["deaths"]
        survival_ticks = episode_log["ticks_alive"]
        raw = (kills * 1.0 - deaths * 0.8 + survival_ticks * 0.002)

    # Latency penalty — applied across all games
    avg_latency = mean(episode_log["tick_latencies"])
    latency_penalty = max(0, (avg_latency - 150) / 150) * 0.2

    return max(0, raw - latency_penalty)

The Critical IM Design Question — Determinism
The biggest gotcha: validators send a challenge to miners in a style that miners cannot easily predict or pre-cache. ￼ For game AI this means episode seeds must be unpredictable in advance. Miners cannot pre-compute “what’s the best move for seed 4821 in pacman” — the seed is only revealed at the start of each tick. Use block hash + validator hotkey + tick number as the seed derivation so miners have no lookahead.

const canvas = document.getElementById('game');
const ctx = canvas.getContext('2d');

// Responsive sizing
const CELL = 20;
const COLS = 21;
const ROWS = 23;
canvas.width = COLS * CELL;
canvas.height = ROWS * CELL;

// ---- TILE TYPES ----
const T = { WALL:0, DOT:1, POWER:2, EMPTY:3, TUNNEL:4, GHOST_HOME:5 };

// ---- COLORS ----
const GHOST_COLORS = ['#ff4466','#22ddff','#ff9922','#cc44ff'];
const GHOST_NAMES  = ['VECTOR','NEXUS','PULSE','DRIFT'];

// ---- GAME STATE ----
let G = {};

function makeInitialState() {
  return {
    score: 0,
    lives: 3,
    level: 1,
    paused: false,
    running: false,
    dead: false,
    dotsLeft: 0,
    // metrics
    deaths: 0,
    pelletsEaten: 0,
    timeStart: Date.now(),
    ghostsCaught: 0,
    escaped: 0,
    closeCallTicks: 0,
  };
}

// ---- MAZE GENERATION ----
// We generate a symmetric maze using recursive division, then overlay
// a fixed ghost-house in the center.

function buildMaze(seed) {
  const rng = seededRng(seed);
  const grid = [];
  for (let r = 0; r < ROWS; r++) {
    grid.push(new Array(COLS).fill(T.WALL));
  }

  // carve passages using randomized DFS
  function carve(r, c) {
    const dirs = shuffle([[0,2],[0,-2],[2,0],[-2,0]], rng);
    for (const [dr, dc] of dirs) {
      const nr = r + dr, nc = c + dc;
      if (nr > 0 && nr < ROWS-1 && nc > 0 && nc < COLS-1 && grid[nr][nc] === T.WALL) {
        grid[r + dr/2][c + dc/2] = T.DOT;
        grid[nr][nc] = T.DOT;
        carve(nr, nc);
      }
    }
  }

  // Start carving from (1,1)
  grid[1][1] = T.DOT;
  carve(1, 1);

  // Mirror left→right for symmetry
  for (let r = 0; r < ROWS; r++) {
    for (let c = 0; c < Math.floor(COLS/2); c++) {
      grid[r][COLS-1-c] = grid[r][c];
    }
  }

  // Ensure border walls
  for (let r = 0; r < ROWS; r++) {
    grid[r][0] = T.WALL;
    grid[r][COLS-1] = T.WALL;
  }
  for (let c = 0; c < COLS; c++) {
    grid[0][c] = T.WALL;
    grid[ROWS-1][c] = T.WALL;
  }

  // Ghost house in center
  const hc = Math.floor(COLS/2), hr = Math.floor(ROWS/2);
  for (let r = hr-2; r <= hr+1; r++)
    for (let c = hc-3; c <= hc+3; c++)
      grid[r][c] = T.GHOST_HOME;
  // Door
  grid[hr-2][hc-1] = T.EMPTY;
  grid[hr-2][hc]   = T.EMPTY;
  grid[hr-2][hc+1] = T.EMPTY;

  // Tunnels on left/right edges row ROWS/2
  const tunnelRow = Math.floor(ROWS/2) + 1;
  grid[tunnelRow][0] = T.TUNNEL;
  grid[tunnelRow][COLS-1] = T.TUNNEL;
  // Make sure tunnel path connects
  for (let c = 0; c < COLS; c++) grid[tunnelRow][c] = c === 0 || c === COLS-1 ? T.TUNNEL : T.DOT;

  // Place power pellets in corners-ish
  const powerSpots = [
    [1,1],[1,COLS-2],[ROWS-2,1],[ROWS-2,COLS-2]
  ];
  for (const [r,c] of powerSpots) {
    if (grid[r][c] === T.DOT) grid[r][c] = T.POWER;
  }

  // Open space around player start
  const pr = ROWS-3, pc = Math.floor(COLS/2);
  for (let dr = -1; dr <= 1; dr++)
    for (let dc = -1; dc <= 1; dc++)
      if (grid[pr+dr]?.[pc+dc] === T.WALL) grid[pr+dr][pc+dc] = T.DOT;

  return grid;
}

// ---- MAZE STATE (dots) ----
let mazeBase = [];  // original tile types (never mutated)
let maze = [];      // live (dots eaten, etc.)

function initMaze(seed) {
  mazeBase = buildMaze(seed);
  maze = mazeBase.map(row => [...row]);
  G.dotsLeft = 0;
  for (let r = 0; r < ROWS; r++)
    for (let c = 0; c < COLS; c++)
      if (maze[r][c] === T.DOT || maze[r][c] === T.POWER) G.dotsLeft++;
}

// ---- PLAYER ----
let player = {};
function initPlayer() {
  player = {
    r: ROWS-3, c: Math.floor(COLS/2),
    px: 0, py: 0,  // sub-cell pixel position
    dir: {dr:0, dc:0},
    nextDir: {dr:0, dc:-1},
    speed: 2.0,      // pixels per frame (increases with level)
    mouthAngle: 0,
    mouthDir: 1,
    moving: false,
    invincible: 0,
    // precise float position
    fr: ROWS-3, fc: Math.floor(COLS/2),
  };
}

// ---- GHOSTS ----
let ghosts = [];
const GHOST_HOME_R = Math.floor(ROWS/2);
const GHOST_HOME_C = Math.floor(COLS/2);

// MODES
const MODE_SCATTER = 'scatter';
const MODE_CHASE   = 'chase';
const MODE_FRIGHT  = 'fright';
const MODE_EATEN   = 'eaten';
const MODE_HOME    = 'home';
const MODE_LEAVE   = 'leave';

// Ghost AI — this is the "miner baseline" script
// Miners replace the chase() function per ghost via the subnet
function ghostAI_baseline(ghost, player, maze, level) {
  const intensity = Math.min(level / 5, 1.0); // ramps 0→1 over 5 levels
  switch (ghost.mode) {
    case MODE_CHASE:   return ai_chase(ghost, player, maze, intensity, ghost.id);
    case MODE_SCATTER: return ai_scatter(ghost);
    case MODE_FRIGHT:  return ai_fright(ghost, player, maze);
    case MODE_EATEN:   return ai_return(ghost);
    case MODE_HOME:    return {dr:0, dc:0};
    case MODE_LEAVE:   return ai_leave(ghost);
  }
}

function ai_chase(ghost, player, maze, intensity, id) {
  // Each ghost has a distinct chase strategy (classic archetypes, renamed)
  // VECTOR (0) — direct BFS to player
  // NEXUS (1) — ambush: target 4 ahead of player
  // PULSE (2) — flanking
  // DRIFT (3) — random + attracted when far
  let target;
  if (id === 0) {
    target = {r: player.r, c: player.c};
  } else if (id === 1) {
    target = {r: player.r + player.dir.dr*4, c: player.c + player.dir.dc*4};
    target.r = clamp(target.r, 0, ROWS-1);
    target.c = clamp(target.c, 0, COLS-1);
  } else if (id === 2) {
    const dx = player.c - ghost.c, dy = player.r - ghost.r;
    const dist = Math.sqrt(dx*dx + dy*dy);
    if (dist > 8 * (1 - intensity*0.5)) {
      target = {r: player.r, c: player.c};
    } else {
      target = ai_scatter(ghost);
    }
    return bfsDir(ghost, target, maze);
  } else {
    // Drift: random with bias toward player at high intensity
    if (Math.random() < intensity) {
      target = {r: player.r, c: player.c};
    } else {
      // random open neighbor
      const opts = openNeighbors(ghost.r, ghost.c, maze)
        .filter(n => n.dr !== -ghost.dir.dr || n.dc !== -ghost.dir.dc);
      if (opts.length) return opts[Math.floor(Math.random()*opts.length)];
      return ghost.dir;
    }
  }
  return bfsDir(ghost, target, maze);
}

function ai_scatter(ghost) {
  const corners = [
    {r:1, c:1}, {r:1, c:COLS-2}, {r:ROWS-2, c:1}, {r:ROWS-2, c:COLS-2}
  ];
  return bfsDir(ghost, corners[ghost.id % 4], maze);
}

function ai_fright(ghost, player, maze) {
  // Run away from player
  const opts = openNeighbors(ghost.r, ghost.c, maze)
    .filter(n => n.dr !== -ghost.dir.dr || n.dc !== -ghost.dir.dc);
  if (!opts.length) return ghost.dir;
  // pick direction that maximizes distance from player
  opts.sort((a,b) => {
    const da = dist2(ghost.r+a.dr, ghost.c+a.dc, player.r, player.c);
    const db = dist2(ghost.r+b.dr, ghost.c+b.dc, player.r, player.c);
    return db - da;
  });
  return opts[0];
}

function ai_return(ghost) {
  return bfsDir(ghost, {r: GHOST_HOME_R, c: GHOST_HOME_C}, maze);
}

function ai_leave(ghost) {
  // Move toward exit
  return bfsDir(ghost, {r: GHOST_HOME_R-2, c: GHOST_HOME_C}, maze);
}

// BFS pathfinding
function bfsDir(ghost, target, maze) {
  const visited = new Set();
  const queue = [{r:ghost.r, c:ghost.c, dir:null}];
  visited.add(`${ghost.r},${ghost.c}`);
  while (queue.length) {
    const {r, c, dir} = queue.shift();
    if (r === target.r && c === target.c) return dir || ghost.dir;
    for (const n of openNeighbors(r, c, maze)) {
      const key = `${r+n.dr},${c+n.dc}`;
      if (!visited.has(key)) {
        visited.add(key);
        queue.push({r:r+n.dr, c:c+n.dc, dir: dir || n});
      }
    }
  }
  // fallback: random open
  const opts = openNeighbors(ghost.r, ghost.c, maze);
  return opts.length ? opts[Math.floor(Math.random()*opts.length)] : {dr:0, dc:0};
}

function openNeighbors(r, c, maze) {
  const dirs = [{dr:-1,dc:0},{dr:1,dc:0},{dr:0,dc:-1},{dr:0,dc:1}];
  return dirs.filter(({dr,dc}) => {
    const nr = r+dr, nc = c+dc;
    if (nr<0||nr>=ROWS||nc<0||nc>=COLS) return false;
    const t = maze[nr][nc];
    return t !== T.WALL;
  });
}

function initGhosts(level) {
  const speed = 1.4 + level * 0.18;
  ghosts = GHOST_NAMES.map((name, i) => ({
    id: i, name,
    r: GHOST_HOME_R + (i < 2 ? 0 : 1),
    c: GHOST_HOME_C + (i % 2 === 0 ? -1 : 1),
    fr: GHOST_HOME_R + (i < 2 ? 0 : 1),
    fc: GHOST_HOME_C + (i % 2 === 0 ? -1 : 1),
    dir: {dr:0, dc: i%2===0?-1:1},
    mode: i === 0 ? MODE_LEAVE : MODE_HOME,
    frightened: false,
    frightenTimer: 0,
    homeTimer: i * 120,  // stagger release
    speed: speed * (0.9 + i*0.05),
    eaten: false,
    blinkPhase: 0,
  }));
}

// Mode schedule (frames)
let modeTimer = 0;
let globalMode = MODE_SCATTER;
const MODE_SCHEDULE = [
  {mode: MODE_SCATTER, dur: 420},
  {mode: MODE_CHASE,   dur: 1200},
  {mode: MODE_SCATTER, dur: 300},
  {mode: MODE_CHASE,   dur: 1200},
  {mode: MODE_SCATTER, dur: 240},
  {mode: MODE_CHASE,   dur: 99999},
];
let modeIdx = 0;

// ---- POWER PELLET STATE ----
let frightenTimer = 0;
const FRIGHTEN_DUR = 360; // frames at level 1, decreases

// ---- ANIMATION LOOP ----
let animId = null;
let lastTime = 0;
const TARGET_FPS = 60;
const FRAME_MS = 1000 / TARGET_FPS;

function gameLoop(ts) {
  animId = requestAnimationFrame(gameLoop);
  const dt = ts - lastTime;
  if (dt < FRAME_MS * 0.8) return;
  lastTime = ts;
  if (!G.running || G.paused || G.dead) return;
  update();
  draw();
}

// ---- UPDATE ----
let frameCount = 0;

function update() {
  frameCount++;

  // Mode schedule
  modeTimer++;
  const sched = MODE_SCHEDULE[modeIdx];
  if (modeTimer >= sched.dur && modeIdx < MODE_SCHEDULE.length-1) {
    modeIdx++;
    globalMode = MODE_SCHEDULE[modeIdx].mode;
    modeTimer = 0;
    // Switch non-frightened, non-eaten, non-home ghosts
    for (const g of ghosts) {
      if (g.mode !== MODE_FRIGHT && g.mode !== MODE_EATEN && g.mode !== MODE_HOME && g.mode !== MODE_LEAVE) {
        g.mode = globalMode;
      }
    }
  }

  // Frighten timer
  if (frightenTimer > 0) {
    frightenTimer--;
    if (frightenTimer === 0) {
      for (const g of ghosts) {
        if (g.mode === MODE_FRIGHT) g.mode = globalMode;
      }
    }
  }

  // Ghost home timers
  for (const g of ghosts) {
    if (g.mode === MODE_HOME) {
      g.homeTimer--;
      if (g.homeTimer <= 0) g.mode = MODE_LEAVE;
    }
  }

  // Move player
  movePlayer();

  // Move ghosts
  for (const g of ghosts) {
    moveGhost(g);
  }

  // Check dot collection
  const pr = Math.round(player.fr), pc = Math.round(player.fc);
  if (pr >= 0 && pr < ROWS && pc >= 0 && pc < COLS) {
    if (maze[pr][pc] === T.DOT) {
      maze[pr][pc] = T.EMPTY;
      G.score += 10;
      G.dotsLeft--;
      G.pelletsEaten++;
      if (G.dotsLeft === 0) triggerLevelComplete();
    } else if (maze[pr][pc] === T.POWER) {
      maze[pr][pc] = T.EMPTY;
      G.score += 50;
      G.dotsLeft--;
      G.pelletsEaten++;
      triggerFrighten();
      if (G.dotsLeft === 0) triggerLevelComplete();
    }
  }

  // Collision detection
  checkCollisions();

  // Update metrics
  updateMetricsDisplay();

  // Player invincibility frames
  if (player.invincible > 0) player.invincible--;

  // Update ghost mouth / animations
  player.mouthAngle += player.mouthDir * 4;
  if (player.mouthAngle > 35) player.mouthDir = -1;
  if (player.mouthAngle < 2)  player.mouthDir = 1;
}

function movePlayer() {
  // Try to turn in nextDir
  const nr = Math.round(player.fr) + player.nextDir.dr;
  const nc = Math.round(player.fc) + player.nextDir.dc;
  if (canMove(nr, nc)) {
    player.dir = {...player.nextDir};
  }

  const targetR = Math.round(player.fr) + player.dir.dr;
  const targetC = Math.round(player.fc) + player.dir.dc;
  const speed = (1.5 + G.level * 0.1) / CELL;

  if (canMove(targetR, targetC) || (player.dir.dr === 0 && player.dir.dc === 0)) {
    player.fr += player.dir.dr * speed;
    player.fc += player.dir.dc * speed;
    player.moving = true;
  } else {
    player.moving = false;
  }

  // Tunnel wrap
  if (player.fc < 0) player.fc = COLS - 1;
  if (player.fc >= COLS) player.fc = 0;
  player.r = Math.round(player.fr);
  player.c = Math.round(player.fc);
}

function moveGhost(g) {
  if (g.mode === MODE_HOME) return;

  const speed = g.speed / CELL * (g.mode === MODE_FRIGHT ? 0.55 : g.mode === MODE_EATEN ? 1.8 : 1.0);

  // At cell center, pick new direction
  const cr = Math.round(g.fr), cc = Math.round(g.fc);
  if (Math.abs(g.fr - cr) < speed * 1.2 && Math.abs(g.fc - cc) < speed * 1.2) {
    g.fr = cr; g.fc = cc; g.r = cr; g.c = cc;
    const newDir = ghostAI_baseline(g, player, maze, G.level);
    if (newDir) {
      const nr = cr + newDir.dr, nc = cc + newDir.dc;
      if (nr>=0 && nr<ROWS && nc>=0 && nc<COLS && maze[nr][nc] !== T.WALL) {
        g.dir = newDir;
      }
    }

    // Check if returned home
    if (g.mode === MODE_EATEN && cr === GHOST_HOME_R && cc === GHOST_HOME_C) {
      g.mode = globalMode;
      g.eaten = false;
    }
    // Check if left home
    if (g.mode === MODE_LEAVE && cr === GHOST_HOME_R-2 && Math.abs(cc - GHOST_HOME_C) <= 1) {
      g.mode = globalMode;
    }
  }

  g.fr += g.dir.dr * speed;
  g.fc += g.dir.dc * speed;

  // Tunnel wrap
  if (g.fc < 0) g.fc = COLS - 1;
  if (g.fc >= COLS) g.fc = 0;
  g.r = Math.round(g.fr);
  g.c = Math.round(g.fc);
}

function canMove(r, c) {
  if (r < 0 || r >= ROWS || c < 0 || c >= COLS) return true; // tunnel
  return maze[r][c] !== T.WALL && maze[r][c] !== T.GHOST_HOME;
}

function checkCollisions() {
  if (player.invincible > 0) return;
  for (const g of ghosts) {
    if (g.mode === MODE_EATEN) continue;
    const dr = Math.abs(g.fr - player.fr);
    const dc = Math.abs(g.fc - player.fc);
    if (dr < 0.7 && dc < 0.7) {
      if (g.mode === MODE_FRIGHT) {
        eatGhost(g);
      } else {
        killPlayer();
        return;
      }
    } else if (dr < 1.2 && dc < 1.2) {
      G.closeCallTicks++;
    }
  }
}

let ghostEatCombo = 0;
const EAT_SCORES = [200,400,800,1600];

function eatGhost(g) {
  g.mode = MODE_EATEN;
  g.eaten = true;
  const pts = EAT_SCORES[Math.min(ghostEatCombo, 3)];
  ghostEatCombo++;
  G.score += pts;
  G.ghostsCaught++;
  showFloatingScore(g.fc, g.fr, '+'+pts, GHOST_COLORS[g.id]);
}

function killPlayer() {
  G.lives--;
  G.deaths++;
  G.escaped++; // ghost escaped (caught player)
  G.dead = true;
  player.invincible = 180;

  const sub = G.lives > 0
    ? `Lives remaining: ${G.lives}\nGhost caught you!`
    : '';

  if (G.lives <= 0) {
    setTimeout(() => showGameOver(), 600);
  } else {
    document.getElementById('dead-title').textContent = 'CONSUMED!';
    document.getElementById('dead-sub').textContent = `Lives remaining: ${G.lives}`;
    document.getElementById('overlay-dead').classList.remove('hidden');
  }
  updateHUD();
}

function triggerFrighten() {
  const dur = Math.max(120, FRIGHTEN_DUR - G.level * 30);
  frightenTimer = dur;
  ghostEatCombo = 0;
  for (const g of ghosts) {
    if (g.mode !== MODE_EATEN && g.mode !== MODE_HOME && g.mode !== MODE_LEAVE) {
      g.mode = MODE_FRIGHT;
    }
  }
}

function triggerLevelComplete() {

  document.getElementById('level-sub').textContent =

}

// ---- FUN SCORE (this feeds miner weights) ----
function calcFunScore() {
  const elapsed = (Date.now() - G.timeStart) / 1000;
  // Danger metric: close calls per second
  const danger = Math.min(G.closeCallTicks / Math.max(elapsed, 1) * 10, 40);
  // Efficiency: pellets per second
  const efficiency = Math.min(G.pelletsEaten / Math.max(elapsed, 1) * 5, 30);
  // Survival: didn't die too much
  const survival = Math.max(0, 30 - G.deaths * 10);
  return Math.round(danger + efficiency + survival);
}

// ---- DRAW ----
function draw() {
  ctx.fillStyle = '#04040a';
  ctx.fillRect(0, 0, canvas.width, canvas.height);

  drawMaze();
  drawDots();
  drawGhosts();
  drawPlayer();
  drawFloats();
}

function drawMaze() {
  for (let r = 0; r < ROWS; r++) {
    for (let c = 0; c < COLS; c++) {
      if (mazeBase[r][c] === T.WALL) {
        // Wall fill
        ctx.fillStyle = '#0a0a33';
        ctx.fillRect(c*CELL, r*CELL, CELL, CELL);

        // Glow outline
        ctx.strokeStyle = '#2222cc';
        ctx.lineWidth = 1.5;
        drawWallSegment(r, c);
      } else if (mazeBase[r][c] === T.GHOST_HOME) {
        ctx.fillStyle = '#0d0d1e';
        ctx.fillRect(c*CELL, r*CELL, CELL, CELL);
      }
    }
  }
  // Ghost house door highlight
  const hr = GHOST_HOME_R-2, hc = GHOST_HOME_C;
  ctx.fillStyle = '#ffaaff44';
  ctx.fillRect((hc-1)*CELL, hr*CELL, 3*CELL, 2);
}

function drawWallSegment(r, c) {
  const x = c*CELL, y = r*CELL;
  const s = CELL;
  // Only draw borders adjacent to non-wall cells for cleaner look
  const adj = (dr, dc) => {
    const nr = r+dr, nc = c+dc;
    if (nr<0||nr>=ROWS||nc<0||nc>=COLS) return false;
    return mazeBase[nr][nc] !== T.WALL;
  };
  ctx.beginPath();
  if (adj(-1,0)) { ctx.moveTo(x,y); ctx.lineTo(x+s,y); }
  if (adj(1,0))  { ctx.moveTo(x,y+s); ctx.lineTo(x+s,y+s); }
  if (adj(0,-1)) { ctx.moveTo(x,y); ctx.lineTo(x,y+s); }
  if (adj(0,1))  { ctx.moveTo(x+s,y); ctx.lineTo(x+s,y+s); }
  ctx.stroke();
}

function drawDots() {
  for (let r = 0; r < ROWS; r++) {
    for (let c = 0; c < COLS; c++) {
      const t = maze[r][c];
      const cx2 = c*CELL + CELL/2, cy2 = r*CELL + CELL/2;
      if (t === T.DOT) {
        ctx.fillStyle = '#ffeeaa';
        ctx.beginPath();
        ctx.arc(cx2, cy2, 2, 0, Math.PI*2);
        ctx.fill();
      } else if (t === T.POWER) {
        const pulse = (Math.sin(frameCount * 0.15) + 1) * 0.5;
        const rad = 4 + pulse * 2;
        ctx.shadowColor = '#ffffff';
        ctx.shadowBlur = 10 + pulse * 10;
        ctx.fillStyle = '#ffffff';
        ctx.beginPath();
        ctx.arc(cx2, cy2, rad, 0, Math.PI*2);
        ctx.fill();
        ctx.shadowBlur = 0;
      }
    }
  }
}

function drawPlayer() {
  const x = player.fc * CELL + CELL/2;
  const y = player.fr * CELL + CELL/2;
  const r = CELL * 0.44;

  // Blink when invincible
  if (player.invincible > 0 && Math.floor(player.invincible / 8) % 2 === 0) return;

  // Determine angle based on direction
  let angle = 0;
  if (player.dir.dc === 1)  angle = 0;
  if (player.dir.dc === -1) angle = Math.PI;
  if (player.dir.dr === -1) angle = -Math.PI/2;
  if (player.dir.dr === 1)  angle = Math.PI/2;

  const mRad = (player.mouthAngle / 180) * Math.PI;

  ctx.shadowColor = '#f7d020';
  ctx.shadowBlur = 12;
  ctx.fillStyle = '#f7d020';
  ctx.beginPath();
  ctx.moveTo(x, y);
  ctx.arc(x, y, r, angle + mRad, angle + Math.PI*2 - mRad);
  ctx.closePath();
  ctx.fill();
  ctx.shadowBlur = 0;

  // Eye
  const eyeX = x + Math.cos(angle - Math.PI/4) * r * 0.45;
  const eyeY = y + Math.sin(angle - Math.PI/4) * r * 0.45;
  ctx.fillStyle = '#04040a';
  ctx.beginPath();
  ctx.arc(eyeX, eyeY, 2, 0, Math.PI*2);
  ctx.fill();
}

function drawGhosts() {
  for (const g of ghosts) {
    if (g.mode === MODE_HOME && g.homeTimer > 180) continue;
    drawGhost(g);
  }
}

function drawGhost(g) {
  const x = g.fc * CELL + CELL/2;
  const y = g.fr * CELL + CELL/2;
  const r = CELL * 0.42;

  let col;
  if (g.mode === MODE_FRIGHT) {
    const blinking = frightenTimer < 120 && Math.floor(frightenTimer/10)%2;
    col = blinking ? '#ffffff' : '#2244ff';
  } else if (g.mode === MODE_EATEN) {
    col = '#ffffff44';
  } else {
    col = GHOST_COLORS[g.id];
  }

  ctx.shadowColor = col;
  ctx.shadowBlur = g.mode === MODE_EATEN ? 0 : 8;
  ctx.fillStyle = col;

  // Ghost body
  ctx.beginPath();
  ctx.arc(x, y - r*0.1, r, Math.PI, 0);
  // Wavy bottom
  const steps = 3;
  const sw = (r*2) / steps;
  for (let i = 0; i <= steps; i++) {
    const bx = x - r + i * sw;
    const by = y + r * 0.9 + (i%2===0 ? -r*0.3 : r*0.1);
    i === 0 ? ctx.lineTo(bx, by) : ctx.lineTo(bx, by);
  }
  ctx.lineTo(x - r, y - r*0.1);
  ctx.fill();
  ctx.shadowBlur = 0;

  // Eyes
  if (g.mode !== MODE_FRIGHT || frightenTimer < 120 && Math.floor(frightenTimer/10)%2) {
    if (g.mode === MODE_FRIGHT) {
      // X eyes when frightened
      ctx.strokeStyle = '#ffffff';
      ctx.lineWidth = 1;
      for (let i = 0; i < 2; i++) {
        const ex = x - 5 + i*10;
        ctx.beginPath();
        ctx.moveTo(ex-3, y-5); ctx.lineTo(ex+3, y-1);
        ctx.moveTo(ex+3, y-5); ctx.lineTo(ex-3, y-1);
        ctx.stroke();
      }
    } else {
      for (let i = 0; i < 2; i++) {
        const ex = x - 5 + i*10;
        ctx.fillStyle = '#ffffff';
        ctx.beginPath(); ctx.arc(ex, y-3, 3.5, 0, Math.PI*2); ctx.fill();
        ctx.fillStyle = '#0011ff';
        const px2 = ex + g.dir.dc*2;
        const py2 = (y-3) + g.dir.dr*2;
        ctx.beginPath(); ctx.arc(px2, py2, 2, 0, Math.PI*2); ctx.fill();
      }
    }
  }
}

function drawFloats() {
  for (let i = floats.length-1; i >= 0; i--) {
    const f = floats[i];
    f.life--;
    f.fr -= 0.3;
    if (f.life <= 0) { floats.splice(i,1); continue; }
    ctx.globalAlpha = f.life / 60;
    ctx.fillStyle = f.color;
    ctx.font = '7px "Press Start 2P"';
    ctx.textAlign = 'center';
    ctx.fillText(f.text, f.fc * CELL + CELL/2, f.fr * CELL);
    ctx.globalAlpha = 1;
  }
}

  // Lives icons (drawn as mini pac-mans)

    cx2.fill();

      <span class="ghost-name">${g.name}</span>
      <span class="ghost-mode">${g.mode.slice(0,3).toUpperCase()}</span>
    `;
    roster.appendChild(div);
  }
}

function seededRng(seed) {
  let s = seed;
  return () => {
    s = (s * 1664525 + 1013904223) & 0xffffffff;
    return (s >>> 0) / 0xffffffff;
  };
}

function shuffle(arr, rng) {
  const a = [...arr];
  for (let i = a.length-1; i > 0; i--) {
    const j = Math.floor(rng() * (i+1));
    [a[i],a[j]] = [a[j],a[i]];
  }
  return a;
}

function clamp(v, lo, hi) { return Math.max(lo, Math.min(hi, v)); }
function dist2(r1,c1,r2,c2) { return (r1-r2)**2 + (c1-c2)**2; }
